# DG4202 任意波形命令确认表

此表用于 v0.4 前置确认。任意波形上传必须先确认 SCPI 命令，再写 driver。

## 当前状态

已确认：

- `*IDN?` 可通过 PyVISA/NI-VISA 访问 DG4202。
- 普通信号控制已在 WaveBench 中实现：function、frequency、Vpp、output、square duty。
- DG4000 系列属于函数 / 任意波形发生器，硬件类别支持任意波形。

未确认：

- 任意波形上传命令。
- 上传数据格式。
- 用户波形存储 / 选择命令。
- 播放采样率或频率语义。

## 命令候选，不可直接当作已确认命令

以下只是其他任意波形发生器或 RIGOL 系列中常见的命名方向，不能直接写入正式 driver：

```text
DATA:DAC
SOURce<n>:DATA
SOURce<n>:FUNC:USER
SOURce<n>:FUNC ARB
SOURce<n>:APPLy:USER
VOLATILE
```

## 安全 probe 规则

- probe 前先读取并保存原状态。
- 默认不要打开输出。
- 如果必须输出，使用低风险参数：1 kHz、1 Vpp、0 V offset。
- 输出端接示波器，不接未知电路。
- probe 后恢复原状态。
- 每个候选命令后立刻读取 `SYST:ERR?`。

## 最小成功定义

v0.4 任意波形上传 MVP 只有在下面闭环成立后才算成功：

```text
上传 waveform -> 选择 waveform -> 输出到 DG4202 CH2 -> RTM2032 CH1 捕获 -> report / FFT 看到预期主频与波形形状
```
